Comandos Git - Guia Rápido

1. Configurando o Nome e E-mail

Antes de fazer commits no Git, você precisa configurar seu nome e e-mail.

git config --global user.name "Seu Nome"
git config --global user.email "seuemail@exemplo.com"

2. Verificando o Status

Use o comando git status para ver quais arquivos foram modificados ou estão prontos para o commit.

git status

3. Adicionando Arquivos ao Staging

Para preparar um arquivo para o commit, você precisa adicioná-lo ao "staging area" com git add.

git add arquivo.txt

Para adicionar todos os arquivos modificados de uma vez:

git add .

4. Fazendo o Commit

Depois de adicionar os arquivos ao staging, use o comando git commit para registrar as mudanças no histórico do projeto.

git commit -m "Mensagem do commit"

5. Enviando para o Repositório Remoto

Use o comando git push para enviar suas mudanças para o repositório remoto, como o GitHub.

git push

6. Recuperando Mudanças do Repositório Remoto

Use o comando git pull para baixar e mesclar as mudanças do repositório remoto.

git pull
