Atividade 1 - O que é uma regra CSS?

Uma regra CSS é um conjunto de instruções usado para estilização de uma pagina HTML ,ela que dar vida e beleza para o conteudo da pagina.

Uma regra CSS é formada por:

Seletor: que indica qual elemento HTML será estilizado

Propriedade: define o que será modificado (cor, tamanho, fonte, etc.)

Valor: especifica como a propriedade será aplicada

Exemplo de uma regra CSS:

p { color: red; font-size: 18px; }

Explicação:

Seletor: p

Propriedades e valores:

  • color: red; deixa o texto vermelho
  • font-size: 18px; aumenta o tamanho da fonte
