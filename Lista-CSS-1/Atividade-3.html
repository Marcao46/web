Atividade 3 Como podemos incluir CSS em documentos HTML?

Existem três maneiras principais de incluir CSS em documentos HTML:

1. CSS Inline: Usando o atributo "style" diretamente em um elemento HTML para aplicar estilos específicos a esse elemento.

2. CSS Interno: Incluindo um bloco de estilos dentro da seção <head> do documento HTML, usando a tag <style>.

3. CSS Externo: Criando um arquivo escrito CSS separado da pagina atual e vinculando-o ao documento HTML usando a tag <link> dentro da seção <head>.
