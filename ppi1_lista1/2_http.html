O que é o Protocolo HTTP

O HTTP (HyperText Transfer Protocol) é o protocolo que permite a comunicação entre o navegador do usuário (cliente) e o servidor que hospeda um site. Ele define como as mensagens são enviadas e recebidas na Web.

Como funciona

O cliente faz uma requisição (request) ao servidor, e este retorna uma resposta (response). O navegador interpreta essa resposta e exibe o conteúdo.

Exemplo de Request:

GET /index.html HTTP/1.1
Host: www.exemplo.com
User-Agent: Chrome/122.0
Accept: text/html

Exemplo de Response:

HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 532
Date: Sat, 11 Oct 2025 16:00 GMT

Cada campo define um aspecto da comunicação — por exemplo, Content-Type indica o tipo de conteúdo retornado.
